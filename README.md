KinectColorApp
==============

EECS 481 project at the University of Michigan

Key Commands:
0: Red Color
1: Green Color
2: Blue Color
3: Eraser
R: Refresh Screen
B: Change Background
Q: Quit

MakeyMakey Key Commands:
W: Red Color
A: Green Color
S: Blue Color
D: Eraser
F: Refresh Screen
G: Change Background
SPACE: Quit
